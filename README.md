## Descripción
[cite_start]Este proyecto consiste en un blog personal desarrollado con **HTML5**, **CSS3** y **Bootstrap 5**. [cite_start]La interfaz es responsiva y sigue un diseño

## Estructura del Proyecto
* [cite_start]`index.html`: Página de inicio con el listado de artículos en tarjetas
* [cite_start]`about.html`: Información personal y biografía
* [cite_start]`contact.html`: Formulario de contacto con validaciones de HTML5
* [cite_start]`assets/styles.css`: Estilos personalizados para branding y feedback visual

## Instrucciones de Navegación
1. [cite_start]Abrir el archivo `index.html` en cualquier navegador
2. [cite_start]Utilizar la **Navbar** superior para navegar entre las tres secciones principales
3. [cite_start]En la página de contacto, el formulario cambiará el color de los bordes a **verde** si el dato es válido y **rojo** si es inválido
